Tools I’ve Used While Learning

Here are some tools I’ve worked with as part of my learning journey in cybersecurity:

  • Hybrid Analysis – I use this to check how malware behaves inside a sandbox.
  • MalwareBazaar – It’s where I get sample malware to experiment with.
  • NeikidEv – This tool has helped me learn static and dynamic analysis techniques.
  • VirusTotal & VT Graph – I use them for quick checks and to analyze how malware samples are linked together.
  • dnSpy – I use this to inspect .NET malware and analyze the code inside.
  • JetBrains IDE – It makes editing and reverse engineering code a lot easier.
  • YARA Rules – I’m learning how to detect malware based on specific patterns.
